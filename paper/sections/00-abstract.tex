\begin{abstract}
\modelname{} is a compact convolutional network (\textbf{1.11\,M} parameters)
that reaches \textbf{99.73\,\%} top-1 accuracy on the 46-class DHCD Devanagari
benchmark~\cite{acharya2015dhcd}---the highest reported result---at
\textbf{15.6\(\times\)} smaller footprint than prior state-of-the-art.
Exact McNemar tests with Wilson confidence intervals, applied across every
student--teacher pairing, found that \emph{no} configuration achieves a
statistically clear win; every model tested---large teacher ensembles included---converges
to the same \textbf{11-error intrinsic floor}.
DHCD is \textbf{saturated} at current resolution.
Parity with the nearest large-model baseline (17.32\,M parameters;
McNemar \(p=0.345\)) holds \emph{even without knowledge distillation}.
On out-of-domain data, \modelname{} transfers to CMATERdb digit recognition
(zero-shot \textbf{76.6\,\%}; fine-tuned \textbf{97.8\,\%}) and holds up far better than large baselines under
common image corruptions (mean corruption accuracy \textbf{75.7\,\%} vs.\ \textbf{38.7\,\%}).
All artifacts are at~\repourl.
\end{abstract}
